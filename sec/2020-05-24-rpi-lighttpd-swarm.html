Description

This guide explains how to set up a lighttpd webserver using Docker swarm and few raspberry pis. The inspiration for this project comes from this guide, and although my setup is similar there is a few things I did differently that might be of interest.

Equipment

  • Raspberry Pis - I have three Raspberry Pi 4's (Model B). They are quite speedy for my purposes. I got them from BuyaPi.
  • Gigabit Ethernet Network Switch - The Raspberry Pi 4 (Model B) has a Gigabit Ethernet Adapter so this makes sense. However older models have a lower bandwidth so this could be overkill depending on what you've got
  • CAT6 Ethernet cable - Atleast one more than the amount of Pis you have. Mine are 1 ft in length and they are still a bit messy near my router.
  • Raspberry Pi Tower - This tower is nice. It's relatively sturdy and the setup is easy. Plus it's got a few fans.s
